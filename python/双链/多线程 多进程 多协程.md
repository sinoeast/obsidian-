# GIL
python速度慢的原因：
1.动态类型语言，边解释边执行
2.GIL，无法利用多核CPU并发执行

全局解释器锁（英语：Global Interpreter Lock,缩写GIL)
是计算机程序设计语言解释器用于同步线程的一种机制，它使得任何时刻仅有一个线程在执行。
即便在多核心处理器上，使用GIL的解释器也只允许同一时间执行一个线程。
